Reference For HTML Forms

Form Tags

Tag Name Description 
<form><form>
 Form

Used to collect and send information
to an external source

 
<input>
 Input

Used to render a variety of input fields on a
webpage including text fields, checkboxes, buttons, etc.
Input element have a type attribute that determines how it gets rendered to a page.

 
<label><label>
 Label

It provides identification for a specific <input> based
on matching values of the <input> ‘s
id attribute and the <label> ‘s for attribute.

 
<select><select>
 Select

It can be used to create a dropdown list.
A list of choices for the dropdown list can be created
using one or more <option> elements.

 
<textarea><textarea>
 Text Area

It is used when creating a text-box for multi-line input (e.g. a comment section).
The element supports the rows and cols attributes which
determine the height and width, respectively, of the element

 
<datalist><datalist>
 Data List

When using an HTML input, a basic search/autocomplete
functionality can be achieved by pairing an <input> with a <datalist> .
To pair a <input> with a <datalist> the <input> ’s
list value must match the value of the id of the <datalist> .
The datalist element is used to store a list of <option> s

Input Type Attribute

Type Description
Text

This renders a single row input field that users can type text inside.
Password

renders a single row input field which allows the user to type censored text inside the field.
It is used to type in sensitive information.
Number

These input fields allow the user to enter only numbers and a few special characters inside the field
Range

A slider can be created by using the type="range" attribute on an HTML input element.
The range slider will act as a selector between a minimum and a maximum value
Checkbox

When using an HTML input element, the type="checkbox" attribute will render a single checkbox item
Radio

HTML <input> elements can be given a type="radio" attribute that renders a single radio button.
